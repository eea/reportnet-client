    ReportnetClient

    The top-level client. One instance per API key. Obtain a DataflowClient via for_dataflow() to avoid repeating dataflow_id on every call.

    ReportnetClient

    from_keyring classmethod 

    from_keyring(
    dataflow_id: int | str,
    base_url: str | None = None,
    *,
    sandbox: bool = False,
    timeout: float = 30.0,
) -> "ReportnetClient"

    Create a client using the API key stored in the system keychain.

    Parameters:

    Name Type Description Default
    dataflow_id int | str

    The dataflow ID the key was stored under.

    		 required
    base_url str | None

    API base URL. Defaults to :data:SANDBOX_URL when sandbox=True, otherwise :data:PRODUCTION_URL.

    		 None
    sandbox bool

    If True, use the sandbox environment and look up the sandbox key (stored separately from the production key).

    		 False
    timeout float

    HTTP request timeout in seconds.

    		 30.0

    Examples::

    # Production (default)
client = ReportnetClient.from_keyring(1619)

# Sandbox
client = ReportnetClient.from_keyring(1619, sandbox=True)

    ping 

    ping(*, dataflow_id: int) -> bool

    Return True if the API key is valid and the API is reachable.

    Makes a single lightweight GET request. Returns False on auth failure; raises on network errors (so transient connectivity issues surface as exceptions rather than a silent False).

    Example::

    if not client.ping(dataflow_id=1619):
    raise RuntimeError("API key is invalid or has been revoked")

    is_big_dataflow 

    is_big_dataflow(*, dataflow_id: int) -> bool

    Return True if dataflow_id is a BigData (DLT2) dataflow.

    Reads the bigData field from GET /dataflow/v1/{dataflowId}. The dedicated GET /dataflow/private/v1/{dataflowId}/isBigDataflow endpoint looks purpose-built for this but 404s for API-key auth regardless of the dataflow's actual BigData status, so it isn't used here.

    etl_export(
    *,
    dataset_id: int,

    version: int = 4,
) -> JobHandle

    GET /dataset/v{version}/etlExport/{datasetId} — async export.

    v4 (BigData/DLT2, recommended): result is a ZIP of CSVs. v5 (analytics, opt-in): result is a ZIP of Parquet files — same shape as v4, smaller and faster to load; never selected automatically, pass version=5 explicitly. v3 (Citus): result is JSON; requires data_provider_codes (ISO country code).

    connect_interactive

    A one-call, non-raising connect helper for interactive tools (used by the example notebooks) — returns (flow, error_message) instead of raising, so callers can render the error directly in a UI.

    connect_interactive 

    connect_interactive(
    dataflow_id: int,
    *,
    sandbox: bool = False,
    country_code: str | None = None,
) -> tuple[DataflowClient | None, str | None]

    Connect for interactive/notebook use, returning a UI-ready result.

    Loads the API key from the system keychain and builds a :class:DataflowClient scoped to dataflow_id. When country_code is given, further scopes it to that reporter via :meth:DataflowClient.find_reporter. When country_code is omitted, validates the key with a single :meth:DataflowClient.ping call instead (there's no specific reporter to resolve).

    This exists mainly to back the "Connect" cell shared by the example marimo notebooks under notebooks/ — see there for usage — but is generally useful for any interactive tool that wants a one-call, non-raising connect step.

    Parameters:

    Name Type Description Default
    dataflow_id int

    The dataflow to connect to.

    		 required
    sandbox bool

    Use the sandbox environment and sandbox key.

    		 False
    country_code str | None

    ISO 3166-1 alpha-2 code to scope to a specific reporter.

    		 None

    Returns:

    Type Description
    DataflowClient | None

    (flow, error_message). On success, error_message is
    str | None

    None. On failure, flow is None and error_message is
    tuple[DataflowClient | None, str | None]

    a short, human-readable string safe to show directly in a UI
    tuple[DataflowClient | None, str | None]

    callout.

    Example::

    flow, error = reportnet.connect_interactive(1619, country_code="IE")
if error:
    mo.callout(mo.md(error), kind="danger")
else:
    mo.callout(mo.md(f"Connected — provider_id={flow._provider_id}"), kind="success")
